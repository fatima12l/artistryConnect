# artistryConnect
Artistry Connect:

Intro:

Artistry Connect is a platform designed to
Empower Artists.
Providing them with a Digital Space To Showcase and Manage their Creative Portfolios.



Language Used:

C++


IDE Used:

QT Creator 15.0.0 (community)


User Interface:
We used QWigdet for the UI in QT Creator.
